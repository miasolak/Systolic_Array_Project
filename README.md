# Systolic Array Design Project

## Overview
This project implements a parameterized systolic array in Verilog for matrix multiplication.

## Files
- systolic_array.v
- control_logic.v
- PE.v
- systolic_array_tb.v

## Features
- Parameterized design
- Modular processing elements
- Control logic for synchronized data flow
- Verification through testbench

## Tools
- Verilog
- Vivado / simulation tools
- Python (if used for matrix generation)
